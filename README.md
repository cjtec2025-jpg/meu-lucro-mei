# Meu Lucro MEI

Aplicativo simples para ajudar MEIs a controlar ganhos, gastos e visualizar o lucro real do mês.

## Objetivo
Ajudar microempreendedores a entender quanto realmente ganham no mês.

## Problema
Muitos MEIs misturam dinheiro pessoal com dinheiro do negócio e não sabem o lucro real.

## Solução
Um app simples, rápido e prático para registrar entradas, saídas e acompanhar o resultado mensal.

## Funcionalidades iniciais
- Registrar entradas
- Registrar saídas
- Visualizar saldo
- Visualizar lucro mensal
- Ver extrato simples
